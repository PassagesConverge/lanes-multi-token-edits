# Lane's Multi-Token Edits

A Foundry VTT module that automatically syncs status effects across all selected tokens.

## Features

- **Automatic Effect Sync**: Toggle a status effect on any selected token, and it applies to all other selected tokens
- **Works with Linked & Unlinked Actors**: Handles both token types seamlessly  
- **Prevents Cascading**: Smart detection prevents infinite loops
- **Client-Side Only**: Changes only apply to the current user's selected tokens

## Installation

In Foundry VTT, use this manifest URL:
```
https://github.com/PassagesConverge/lanes-multi-token-edits/releases/latest/download/module.json
```

## Usage

1. Select multiple tokens (Ctrl+Click or drag-select)
2. Right-click on any selected token
3. Toggle any status effect
4. Effect automatically applies to all selected tokens
